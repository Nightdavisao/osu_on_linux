# osu_on_linux
An easy osu! installer for Linux platform using the frontend PlayOnLinux
## How to install
1. Download and install PlayOnLinux
2. Download "osu_on_linux.sh" file
3. Open PlayOnLinux and go to "Tools" and click in "Run a local script"
4. Click "Next" and click "Browse" and set the "osu_on_linux.sh"
5. Just follow the installer assistant
6. When you reach to the osu! installing, wait install correctly and when osu! open, exit the game.
7. And follow the installer again and when you finished, you can just open the game doing double-click in the osu! name.
8. (Optional) Create an account in https://osu.ppy.sh/home to download cool beatmaps!
